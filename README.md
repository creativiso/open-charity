# OpenCharity

## Описание на платформата

OpenCharity е платформа, предназначена за публикуване и откриване на кампании или каузи в различни области, градове и организации.

## Development

```bash
# Start all services
yarn services:start

# Start development server
yarn workspace server dev

# Run tests
yarn test

# Linting and formatting
yarn lint
yarn lint:fix
yarn format
```

## Requirements

- Node.js 20+,
- Docker 20.10+,
- Docker Compose V2
