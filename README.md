# OpenCharity

## Описание на платформата 

OpenCharity е платформа, предназначена за публикуване и откриване на кампании или каузи в различни области, градове и организации.

---

## Стартиране на платформата
- инсталиране на dependencies 
```
yarn install
```

- стартиране на server / backend

- стартиране на client / frontend
